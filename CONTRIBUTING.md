# Contributing to Awesome Scraping

Thank you for taking the time to contribute! Your suggestions help make this list a valuable resource for the web scraping community.

Before submitting a repository, please ensure it meets these standards:

- **Relevance:** Must be directly related to web scraping, crawling, browser automation, data parsing, or extraction.
- **Quality:** Should have a clear `README.md` with documentation that makes it useful to others.
- **Activity:** Should have been updated within the last 6-12 months (exceptions made for legendary/foundational tools).
- **Open Source:** Must be an open-source project hosted on GitHub.

---

## How to Contribute

### Adding a New Repository

1. **Check if it already exists** - Search the list to avoid duplicates.
2. **Create an issue** with the title format: `add: https://github.com/owner/repo`
3. **Wait for review** - A maintainer will evaluate your suggestion against the inclusion criteria and add it if approved.

### Submission Checklist

Before submitting, verify that:

- [ ] The issue title starts with `add:` followed by the GitHub URL
- [ ] The repository is not already in the list
- [ ] The GitHub link is valid and accessible
- [ ] The repository meets all inclusion criteria above
